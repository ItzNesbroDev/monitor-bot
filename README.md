• Go to config.json
• replace `YOUR BOT TOKEN HERE` to Your Bot Token
• replace `YOUR BOT PREFIX HERE` to your bot prefix
• replace `LINK LIMTE` To Your Own Link Monitoring limte

• Type: `node index.js` on your console to start
